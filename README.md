  Senior Software Engineer with nearly a decade of professional experience spanning **fintech**, **industrial control systems**, and **computer robotics**. I lead a development team and

  ### What I Bring

  - **Architecture & Leadership** — Lead a development team of 4, driving technical decisions and delivery
  - **Event-Driven Systems** — Migrated a payment collection system from legacy polling to event-driven architecture, eliminating race conditions, improving performance, and opening the possibilities to extend features through those changes
  - **Distributed Systems** — Deep experience with messaging frameworks like NServiceBus, CQRS, and domain-driven design
  - **Robotics & Real-Time Systems** — Spearheaded the development of an indoor navigation and signaling system enabling near real-time multi-robot communication via WebSockets
  - **Industrial Software** — Delivered new versions of distributed control system software with hardware support expansions at Emerson
